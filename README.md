# StudyAI

**StudyAI**, öğrencilerin çalışma süreçlerini optimize etmek, verimliliği artırmak ve akademik başarıya odaklanmalarını sağlamak amacıyla geliştirilmiş, Python ve Flask tabanlı bir çalışma planlama ve yönetim uygulamasıdır.

## 📋 Proje Hakkında
Bu proje, kullanıcıların günlük çalışma rutinlerini düzenlemelerine, görevlerini yönetmelerine ve akademik ilerlemelerini takip etmelerine yardımcı olan bir web platformudur. C# ve Python gibi dillerdeki yetkinliklerimi ve veritabanı yönetimi (Microsoft Access ve SQL) konusundaki tecrübelerimi birleştiren, kullanıcı odaklı bir sistemdir.

## 🚀 Temel Özellikler
* **Görev Yönetimi:** Günlük çalışma görevlerini ekleme, listeleme ve tamamlama.
* **Akademik İzleme:** Çalışma verimliliğini analiz etme ve planlama araçları.
* **Kullanıcı Kimlik Doğrulama:** Güvenli giriş ve oturum yönetimi.
* **Ayarlanabilir Yapı:** Uygulama içi konfigürasyonlar ve kişiselleştirilmiş ayarlar.
* **Sağlık Kontrolleri:** Sistem durumunun izlenmesi.

## 🛠️ Teknik Altyapı
* **Backend:** Python, Flask.
* **Frontend:** HTML5, CSS3, JavaScript.
* **Veritabanı:** SQL / Access altyapısı.
* **API:** RESTful API yapısı.

## 📁 Dosya Yapısı
* `/api`: Backend servisleri, kimlik doğrulama, görev yönetimi ve veritabanı bağlantıları.
* `/assets`: Proje logoları ve grafik kaynaklar.
* `/css`: Stil dosyaları (main.css, animations.css, components.css).
* `/index.html`: Web arayüzünün ana giriş noktası.
* `/database.sql`: Veritabanı şeması.

## 📦 Kurulum ve Çalıştırma
Projeyi yerel ortamınızda çalıştırmak için:

1.  Depoyu klonlayın:
    ```bash
    git clone [repo-url]
    ```
2.  Gerekli Python kütüphanelerini kurun (Flask gereklidir).
3.  `.env` dosyasını oluşturun ve gerekli ortam değişkenlerini tanımlayın.
4.  Uygulamayı başlatın.
